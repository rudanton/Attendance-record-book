## 1. 새로운 매장 PC 인증하기

보안을 위해, 사장님이 **인증한 PC에서만** 출퇴근 기능을 사용할 수 있습니다. 새로운 PC를 매장용으로 등록하려면 다음 절차를 따르세요. (이 과정은 개발자의 도움이 필요할 수 있습니다.)

1.  새로운 PC의 웹 브라우저에서 출퇴근 시스템 주소(`http://localhost:3000` 또는 실제 운영 주소)로 접속합니다.
2.  화면에 **"Access Denied"** 또는 **"접근 거부"** 메시지가 표시됩니다.
3.  이 PC의 **고유 인증 코드(토큰)**를 알아내야 합니다.
    *   **방법**: 개발자에게 "새로운 PC의 `pizza_shop_device_token` 값을 Firestore에 추가해 주세요"라고 요청합니다.
4.  개발자가 해당 코드를 시스템에 등록하면, 새로운 PC에서 새로고침(F5) 시 정상적으로 출퇴근 대시보드가 보이게 됩니다.

## 2. 직원 관리하기

직원을 추가하거나 퇴사 처리하는 모든 관리는 **관리자 페이지**에서 이루어집니다.

*   **관리자 페이지 주소**: 시스템 주소 뒤에 `/admin`을 붙여 접속합니다. (예: `http://localhost:3000/admin`)

### 신규 직원 추가하기

1.  관리자 페이지 상단의 **'Add New Employee'** 섹션으로 이동합니다.
2.  각 칸에 정보를 입력합니다.
    *   **Name**: 직원의 이름을 입력합니다.
    *   **PIN (4 digits)**: 직원이 사용할 4자리 숫자 암호를 입력합니다.
    *   **Hourly Rate**: 직원의 시급을 숫자로만 입력합니다.
3.  **'Add Employee'** 버튼을 누르면 목록에 바로 추가됩니다.

### 기존 직원 퇴사 처리하기 (비활성화)

직원이 퇴사하더라도 데이터는 삭제되지 않고 **비활성 상태**로 안전하게 보관됩니다.

1.  관리자 페이지의 **'Employee List'**에서 퇴사 처리할 직원을 찾습니다.
2.  해당 직원의 상태(Status)가 초록색 'Active'인 것을 확인합니다.
3.  오른쪽 끝에 있는 **'Deactivate'** 버튼을 누릅니다.
4.  "정말로 비활성화하시겠습니까?" 라는 확인 창이 뜨면 '확인'을 누릅니다.
5.  직원의 상태가 빨간색 'Inactive'로 변경되며, 메인 출퇴근 화면에 더 이상 나타나지 않습니다.

### 퇴사한 직원 복귀 처리하기 (재활성화)

과거에 일했던 직원이 다시 돌아온 경우, 간단하게 계정을 복구할 수 있습니다.

1.  관리자 페이지의 **'Employee List'**에서 복귀 처리할 직원을 찾습니다.
2.  해당 직원의 상태(Status)가 빨간색 'Inactive'인 것을 확인합니다.
3.  오른쪽 끝에 있는 **'Reactivate'** 버튼을 누릅니다.
4.  "정말로 재활성화하시겠습니까?" 라는 확인 창이 뜨면 '확인'을 누릅니다.
5.  직원의 상태가 다시 초록색 'Active'로 변경되며, 메인 출퇴근 화면에 즉시 나타납니다.
